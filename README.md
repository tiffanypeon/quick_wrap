# QuickWrap

## Installation

Add this line to your application's Gemfile:

    gem 'QuickWrap'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install QuickWrap

## Usage

    client = QuickWrap::Client.new authorization: 'your secret token', environment: 'stagegi'
