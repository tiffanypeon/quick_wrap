# Jawb

## Installation

Add this line to your application's Gemfile:

    gem 'jawb'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jawb

## Usage

    client = Jawb::Client.new authorization: 'your secret token', environment: 'stagegi'
